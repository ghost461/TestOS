# TestOS
## 基于X86架构的初级操作系统内核
- 启动过程使用GURB引导，遵循multiboot标准(使用1.44MB软盘引导)
- 库函数:
- 	字符模式的显卡驱动(默认80x25大小)
- 	内核打印函数库(printk)
- 	字符串处理(string)
- 	ELF可执行文件支持(elf)
- 保护模式与全局段描述符表
- 中断描述符表
- 物理内存分页式管理(当前仅支持最大512MB)
